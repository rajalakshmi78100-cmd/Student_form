# Student Form

A simple Student Form web application built using **HTML, CSS, and JavaScript**. This project allows users to enter student details through a form and displays the submitted information in a table on the same page.

## Features

- Add student details using a form
- Display submitted data in a table
- User-friendly interface
- Responsive design

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Use

1. Open the application in your browser.
2. Enter the student details in the form.
3. Click the **Save** button.
4. The entered data will be displayed in the table below.
